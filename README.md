# Joshua Jones Portfolio Website

Personal portfolio website for Joshua Jones, a recent computer science graduate and aspiring software engineer.

This website highlights my background, education, employment experience, technical skills, and software development projects.

## Featured Project

### Job Application Tracker

A full-stack job application tracking platform designed to help users manage job applications, interviews, and hiring progress in one centralized system.

The project demonstrates full-stack development skills including:

* React Native and Expo frontend development
* Node.js and Express.js backend development
* REST API design
* MySQL relational database integration
* JWT authentication and protected routes
* Cross-platform mobile and web support
* Reusable UI and form components
* Client-side and server-side validation
